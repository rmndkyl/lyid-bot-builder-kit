## 🚀 Quick Start

### Prerequisites

1. [Hermes Agent](https://hermes-agent.nousresearch.com/docs) (free)
2. [Node.js](https://nodejs.org/) 18+
3. [Git](https://git-scm.com/)

### Installation

```bash
# Clone this repo
git clone https://github.com/rmndkyl/lyid-bot-builder-kit.git
cd lyid-bot-builder-kit

# Copy skills to Hermes
cp -r endpoint-hunter/ ~/.hermes/skills/airdrop-automation/endpoint-hunter/
cp -r script-architect/ ~/.hermes/skills/airdrop-automation/script-architect/
cp -r project-lifecycle-tracker/ ~/.hermes/skills/airdrop-automation/project-lifecycle-tracker/
```

### Usage

Open Hermes Agent and say:

```
Research project ini: https://example.com
```

The skills will:
1. Trace the API endpoints
2. Score the project for automation potential
3. Generate a working bot script
4. Push to GitHub
